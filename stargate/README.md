# Custom Stargate module example

## Build and run

Clone the whole repository containing this project:

```shell
$ git clone https://github.com/DariuszDepta/mt-on-tour.git
```

Change the current directory to **stargate**:

```shell
$ cd stargate
```

Build the project:

```angular2html
$ cargo build
```

Run test example:

```shell
$ cargo test
```
